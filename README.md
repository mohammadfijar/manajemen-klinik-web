Sistem Janji Temu Klinik
DeskripsiSistem Janji Temu Klinik adalah platform berbasis web untuk manajemen janji temu pasien dengan dokter.

Fitur- Login multi-role (admin, dokter, pasien)
- Pembuatan janji temu pasien
- Manajemen jadwal dokter
- Riwayat janji temu pasien

Teknologi- Frontend: HTML, CSS, JavaScript
- Backend:  PHP
- Database: MySQL

Cara Instalasi
1. Clone repository
2. Konfigurasi database
3. Jalankan aplikasi
